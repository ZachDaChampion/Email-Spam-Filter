uint64_t hash::fnv1a_64 ( std::string  data)

An implementation of the 64-bit FNV-1a hash function.

The Fowler-Noll-Vo hash function is simple and efficient algorithm that distributes hashes evenly. For more information, see https://en.wikipedia.org/wiki/Fowler%E2%80%93Noll%E2%80%93Vo_hash_function#FNV-1a_hash

Parameters
dataThe data, in string form, that is to be hashed.
Returns
A 64-bit unsigned integer that represents the hash of the data.

◆ mod10()

uint64_t hash::mod10 ( std::string  data)

A simple hash algorithm used for testing.

Simply sums the characters in data and mods that sum by 10. This algorithm is designed to have many collisions in order to more easily test the hash table. Should not be used for a production hash table.
